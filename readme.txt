== Installation ==

1. Log in to your WordPress dashboard (if the site is multisite, login as a multisite super admin and go to the Network dashboard at /wp-admin/network).
2. Go to Plugins and make sure the free NS Cloner core plugin (version 4.0 or later) is installed (if not you can search "ns cloner" to find it on the WordPress plugin repository).
3. Go to Plugins > Add New tools to install the add-on by uploading its zip file that you downloaded from NeverSettle.it
4. Activate the plugin.
